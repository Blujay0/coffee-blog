# Coffee Blog
Personal playground for trying out different styles and page layouts. <br />

## Features
* HTML
* Semantic HTML
* CSS
* CSS Reset
  * compressed set of CSS rules that *resets* the styling of all HTML elements to a consistent baseline.
* Google Fonts API
* GitHub Page Deployment
